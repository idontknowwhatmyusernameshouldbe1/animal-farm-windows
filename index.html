Photos, names, and the small stories that make them yours — on your PC.

Download for Windows

Zip · Windows 10/11 (64-bit) · No .NET install needed

Install

  1. Download AnimalFarm-Windows.zip.
  2. Unzip the folder anywhere you like.
  3. Open AnimalFarm.exe.

The first launch may show a Windows SmartScreen prompt. Choose More info, then Run anyway if you trust this build.

On this device

Pets and photos stay in %LocalAppData%\AnimalFarm\ on this computer. They are not uploaded anywhere and are separate from the browser web gallery.
